Словари и их методы

Словарь dict – коллекция произвольных объектов с доступом по ключу

Ключ key – строка, число или кортеж

Значение value – любой тип данных

Элемент item – кортеж вида (key, value)

Методы словарей

Создание словаря

1. Классика

Короткий и многообразный словарь можно создать, поместив необходимые элементы в фигурные скобки. Этот способ силён, если необходимо быстро вручную записать несколько элементов. Особенно эффективен, если элементы состоят из разных типов данных. Приведём пример простого словаря, хранящего имена работников в разных офисах компании

offices = {'office 1': ['Igor', 'Nataly'], 'office 2': ['Ann', 'Kirill']}

Очевидный минус этого способа: им невозможно заполнить огромное количество данных

2. Генератор словарей

Предположим, нам надо описать функцию y = x² - 1 для всех целочисленных x на отрезке [1, 5]. Это можно сделать следующим образом:

dots = {x: x ** 2 - 1 for x in range(1, 6)}

#{1: 0, 2: 3, 3: 8, 4: 15, 5: 24}

Теперь в словаре для любой целочисленной абциссы на заданном отрезке будет храниться её ордината, что позволит получить значение функции, просто обратившись к значению словаря

3. Функция dict([other])

Функция создаёт элементы словаря из произвольного количества упорядоченных пар. Если функция принимает на вход словарь, результатом будет тот же словарь, отсортированный по ключам. Приведём пример присвоения идентификационных номеров работникам из разных офисов

workers = dict([("Ann", 1), ("Igor", 2), ("Kirill", 3), ("Nataly", 4)])
workers = dict(Ann=1, Igor=2, Kirill=3, Nataly=4)

#{"Ann": 1, "Igor": 2, "Kirill": 3, "Nataly": 4}

Обращаем внимание на то, что кавычки в последнем из примеров не нужны
